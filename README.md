# Random User API Integration

## Description
A responsive web application that fetches and displays random user data using Fetch API.

## Features
- Profile Photo
- Name
- Email
- Country
- Generate New User Button
- Responsive Design
- Hover Effects

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API

## API Used
https://randomuser.me/api/
